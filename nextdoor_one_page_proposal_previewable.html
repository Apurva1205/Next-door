Metrics Glossary (Nextdoor)

  • Activation: % of new neighbors who complete key setup (e.g., verify address, first follow) within N days.
  • Retention (7-day): % of activated neighbors who return in the following 7 days.
  • Reply Rate: Replies per 100 posts/views; segment by content type & neighborhood tenure.
  • Guardrails: Negative outcomes we control (reports, blocks, mutes) normalized per 1k sessions.
  • CUPED: Variance-reduction technique using pre-period covariates to improve test sensitivity.
  • Diff-in-Diff: Causal estimator for staged rollouts; requires parallel trends checks.
